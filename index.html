In this project an investigation was performed on Front Controllers and their use in web application. This included how the Front Controller pattern can be integrated into MVC (Model-View-Controller) architecture design pattern.

To demonstrate the advantages (and disadvantages) of the MVC design pattern. A Web Application created with the PHP Framework Symfony 2.0 was created.
